Issue: MongoDB not responding
Solution:
  1. Check status: ./mongodb.sh status
  2. View logs: ./mongodb.sh logs
  3. Restart: ./mongodb.sh restart

Issue: Port 27017 already in use
Solution:
  1. Find process: netstat -tlnp | grep 27017
  2. Stop MongoDB: ./mongodb.sh stop
  3. Restart: ./mongodb.sh start

Issue: Connection refused from Django
Solution:
  1. Verify MongoDB is running: ./verify_mongodb.sh
  2. Check connection string in Django settings
  3. Ensure credentials are correct (admin/password)

See MONGODB_SETUP.md for detailed troubleshooting.
